# pmmm-madomagi-archives
Archives of old PMMM stuff.

This directory contains an archive of the official Puella Magi Madoka Magica Character Quiz released back around ~2013 or so.
The original website is no longer available, but all the logic was already present inside the .html file (taken from the Internet Archive), so it was fairly easy to recreate the original behavior of the site.
